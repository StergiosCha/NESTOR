=== COQ SYNTAX RULES ===

Use this setup pattern:

  Parameter Entity : Type.
  (* Declare predicates as Parameters *)
  Parameter man : Entity -> Prop.
  Parameter woman : Entity -> Prop.
  Parameter walk : Entity -> Prop.
  (* Declare relations with two arguments *)
  Parameter love : Entity -> Entity -> Prop.
  (* Declare named individuals *)
  Parameter john mary : Entity.

Quantifiers:
  "every man walks"    -> forall x : Entity, man x -> walk x
  "some man walks"     -> exists x : Entity, man x /\ walk x
  "no man walks"       -> ~ exists x : Entity, man x /\ walk x

Connectives:
  /\  (and)
  \/  (or)
  ~   (not)
  ->  (implies)
  <-> (iff)

=== PROOF INSTRUCTIONS ===

For entailment: prove that the premise implies the hypothesis.
  Theorem nli_result : (* premise -> hypothesis *).
  Proof. (* your proof *) Qed.

For contradiction: prove that the premise implies the negation of the hypothesis.
  Theorem nli_result : (* premise -> ~ hypothesis *).
  Proof. (* your proof *) Qed.

For neutral/unknown: the goal should not be provable. Use Abort.
  Theorem nli_result : (* premise -> hypothesis *).
  Proof. (* attempt *) Abort.
